# Tattoo AI Simulator v4

**Physics-based Tattoo Machine Simulator with Adaptive AI (PID + RL)**

En real-time web-baseret simulator, der modellerer hele tatoveringsprocessen — fra nålevibration og hudinteraktion til blækdeponering og traumehåndtering.

### Features
- Bevægende nål med realistisk vibration (line + circle mode)
- Pseudo-3D hudvisualisering med dybde og belysning
- HUD Visor — live tværsnit af hudlagene (epidermis, dermis, hypodermis)
- Adaptiv AI-styring (PID + Fuzzy Logic + simuleret Reinforcement Learning)
- Fuldt responsivt design — fungerer godt på iPhone, Mac og PC

### Sådan kører du projektet
1. Åbn filen `index.html` i en moderne browser (Chrome, Firefox eller Safari anbefales)
2. Tryk på **START**
3. Brug dropdown-menuen til at skifte mellem "Straight Line" og "Circle"

### Teknologi
- Vanilla HTML5 + Canvas
- Real-time fysiksimulation
- Adaptiv AI-kontrol
- Responsivt design
